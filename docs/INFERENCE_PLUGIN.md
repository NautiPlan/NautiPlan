# tauri-plugin-taskpilot-inference

为 Tauri 2 提供 TaskPilot 推理核心能力的插件：

- 端侧 LLM：非流式推理、流式 token 推送
- Embedding：文本向量化
- RAG：SQLite 知识库的文档入库 / 清空 / Top-K 检索
- 可选：启动本地 OpenAI 风格 HTTP API 服务器（`/v1/chat/completions`，支持 SSE）

底层 C/C++ 接口说明见：`INFERENCE_INTERFACE.md`。

## 平台与构建说明

本仓库包含 Rust 插件 + C++ 推理核心源码，插件的构建脚本会尝试编译并链接 C++ 核心。

- Android：使用 CMake preset 编译，并将构建产物递归复制到 `android/src/main/jniLibs/<abi>`
- Linux：使用 CMake preset 编译并链接
  - 目前仓库主要保证 Android 构建链路可用；其他平台可以先按需补齐 preset/依赖再启用

你可以在 `build.rs` 里看到当前支持的 preset 选择逻辑。

### 依赖

- Rust toolchain（本仓库 `Cargo.toml` 指定 `rust-version = 1.77.2`）
- CMake（`cmake` 可执行文件）
- Android：需要设置 `ANDROID_NDK`（或 `NDK_HOME`/`ANDROID_NDK_HOME`）

> 注意：如果 CMake 配置阶段报 `thirdparty/*` 缺少 `CMakeLists.txt`，通常表示子模块/依赖没拉全，需要先初始化/拉取 thirdparty。

## 在 Tauri App 中集成

### 1) Rust 侧注册插件

在你的 `src-tauri/Cargo.toml` 添加依赖（路径依赖示例）：

```toml
[dependencies]
tauri-plugin-taskpilot-inference = { path = "../tauri-plugin-taskpilot-inference" }
```

在 `src-tauri/src/lib.rs`（或 `main.rs`）注册插件：

```rust
tauri::Builder::default()
	.plugin(tauri_plugin_taskpilot_inference::init())
	.run(tauri::generate_context!())
	.expect("error while running tauri application");
```

### 2) 配置权限（Tauri 2 Capabilities）

插件命令受权限控制。权限标识符可在以下文件查看：

- `permissions/autogenerated/reference.md`
- `permissions/autogenerated/commands/*.toml`

在你的 Tauri App 的 capability 文件里加入允许权限（示例，仅按需开启）：

```json
{
  "identifier": "main",
  "description": "Main capability",
  "windows": ["main"],
  "permissions": [
    "taskpilot-inference:allow-llm-init",
    "taskpilot-inference:allow-llm-chat",
    "taskpilot-inference:allow-llm-chat-stream",
    "taskpilot-inference:allow-llm-release",
    "taskpilot-inference:allow-embedding-init",
    "taskpilot-inference:allow-embedding-encode",
    "taskpilot-inference:allow-rag-init",
    "taskpilot-inference:allow-rag-add-document",
    "taskpilot-inference:allow-rag-retrieve"
  ]
}
```

> 提示：如果你想“全开默认权限”，插件仓库提供了 `permissions/default.toml` 作为参考。

## 前端调用（JavaScript/TypeScript）

Tauri 2 中使用 `invoke` 调用插件命令。命令名格式：

`plugin:taskpilot-inference|<command>`

### LLM：初始化 + 非流式对话

```ts
import { invoke } from "@tauri-apps/api/core";

await invoke("plugin:taskpilot-inference|llm_init", {
  payload: {
    configPath: "models/Qwen2.5-1.5B-Instruct-MNN/opencl-config.json",
  },
});

const { response } = await invoke<{ response: string }>(
  "plugin:taskpilot-inference|llm_chat",
  { payload: { query: "你好，请简单自我介绍。" } }
);

console.log(response);
```

### LLM：流式对话（token 推送）

`llm_chat_stream` 会通过 `Channel` 逐 token 推送：

```ts
import { invoke, Channel } from "@tauri-apps/api/core";

type StreamToken = { token: string; done: boolean };

const onToken = new Channel<StreamToken>();
onToken.onmessage = (msg) => {
  if (msg.done) {
    console.log("\n[DONE]");
  } else {
    // 浏览器环境请改用累加字符串
    process.stdout.write(msg.token);
  }
};

await invoke("plugin:taskpilot-inference|llm_chat_stream", {
  payload: { query: "用一段话描述什么是人工智能。" },
  onToken,
});
```

### Embedding：文本向量化

```ts
import { invoke } from "@tauri-apps/api/core";

await invoke("plugin:taskpilot-inference|embedding_init", {
  payload: { configPath: "models/embedding/config.json" },
});

const { vector, dimension } = await invoke<{
  vector: number[];
  dimension: number;
}>("plugin:taskpilot-inference|embedding_encode", {
  payload: { text: "Testing embedding" },
});

console.log(dimension, vector.slice(0, 5));
```

### RAG：初始化 + 入库 + 检索

RAG 依赖 Embedding：先 `embedding_init`，再 `rag_init`。

```ts
import { invoke } from "@tauri-apps/api/core";

await invoke("plugin:taskpilot-inference|embedding_init", {
  payload: { configPath: "models/embedding/config.json" },
});

await invoke("plugin:taskpilot-inference|rag_init", {
  payload: { dbPath: "data/rag.sqlite" },
});

await invoke("plugin:taskpilot-inference|rag_add_document", {
  payload: {
    text: "TaskPilot 是一个端侧推理框架...",
    chunkSize: 500,
    chunkOverlap: 50,
  },
});

const { context } = await invoke<{ context: string; count: number }>(
  "plugin:taskpilot-inference|rag_retrieve",
  { payload: { query: "什么是 TaskPilot？", topK: 2 } }
);

console.log(context);
```

### 可选：启动本地 HTTP API 服务器

底层 server 是阻塞式 `listen`，插件里用后台线程启动（目前没有 stop 接口）。

```ts
import { invoke } from "@tauri-apps/api/core";

const { url } = await invoke<{ url: string }>(
  "plugin:taskpilot-inference|llm_server_start",
  { payload: { port: 8080 } }
);

console.log("API base:", url);
```

## 命令一览

- LLM
  - `llm_init` / `llm_release` / `llm_status`
  - `llm_chat`（非流式）
  - `llm_chat_stream`（流式 token）
  - `llm_server_start` / `llm_server_url`
- Embedding
  - `embedding_init` / `embedding_release` / `embedding_status`
  - `embedding_encode`
- RAG
  - `rag_init` / `rag_release` / `rag_status`
  - `rag_add_document` / `rag_clear`
  - `rag_retrieve`

## 资源释放建议

插件在进程退出时会尝试释放资源，但建议在不再使用时主动调用：

- `llm_release`
- `embedding_release`
- `rag_release`
