## Default Permission

Default permissions for taskpilot-inference

#### This default permission set includes the following:

- `allow-string-free`
- `allow-llm-init`
- `allow-llm-release`
- `allow-llm-chat`
- `allow-llm-chat-stream`
- `allow-llm-server-start`
- `allow-llm-server-url`
- `allow-llm-status`
- `allow-embedding-init`
- `allow-embedding-release`
- `allow-embedding-encode`
- `allow-embedding-status`
- `allow-rag-init`
- `allow-rag-release`
- `allow-rag-add-document`
- `allow-rag-clear`
- `allow-rag-retrieve`
- `allow-rag-status`
